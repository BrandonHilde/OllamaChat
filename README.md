# OllamaChat
Allows you to talk using Ollama


## How to run apps

1. Install Electron via `npm install -g electron`
2. Install Ollama (https://ollama.com/)
3. Install gemma2 with: `ollama run gemma2`
4. Run with command: `npm start`
